# <h1>SOC-Alert-Incident-Reports<h1>


<h2>Description</h2>
This repository contains a collection of Security Operations Center (SOC) alert investigations performed in a simulated environment. A new report is added daily 

Each report documents a real-world scenario including:
- Alert analysis  
- Investigation steps  
- Indicators of Compromise (IOCs)  
- Remediation actions  
- Final conclusions  

These reports are part of my **hands-on cybersecurity training**, focused on developing practical skills in:
- Threat detection
- Threat Intelligence
- Incident response  
- Log analysis  
- Malware Analysis
- SIEM Operations
- Network Analysis
